PRODUCER-CONSUMER MODEL

This is a proof-concept project.
My goal is to apply several JEE technologies, in a well-known model of producers&consumers.

In this branch, I've created a java desktop application.

TECHNOLOGIES:
 * Maven
 * Spring Framework: Core Module (Beans and IoC Container)
 * Spring Framework: AOP Module
 * Log4j
